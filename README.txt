== Description ==

Portfolio plugin integrates well with your WordPress site. This plug-in allows you to create Portfolio that have the same functionality as posts, while adding further features that allow you to manage your Portfolio by adding image and text [get-post-img show='false'],[recent-posts posts="2"] shortcode.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `portfolio.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `[portfolio-posts]` shortcode in your templates/page
